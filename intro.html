Si Pemban adalah aplikasi demo offline-first untuk mencatat curah hujan (mm/hari), menghitung status risiko (Normal/Waspada/Siaga/Awas), lalu memberi rekomendasi tindakan. Data disimpan di localStorage (tanpa akun & tanpa backend).

Input Data
Tanggal + wilayah + curah hujan + kategori kejadian (Genangan/Banjir/Longsor).
Status Otomatis
Berdasarkan ambang yang bisa kamu atur sendiri di menu Pengaturan.
GPS & Peta
Koordinat opsional (Ambil GPS) lalu tampil sebagai marker di peta.
Export/Import
Export CSV/JSON untuk demo & pindah device.
Status & rekomendasi di aplikasi ini bersifat heuristik untuk demo, bukan keputusan resmi instansi.
Masuk Aplikasi Langsung Tambah Data Atur Ambang Risiko
Tips: Kamu bisa install sebagai PWA dari browser (jika didukung) lalu jalankan seperti aplikasi.
